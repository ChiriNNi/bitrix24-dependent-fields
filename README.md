# Зависимые поля Bitrix24

MVP CRM-приложения для выбора зависимых полей в карточке сделки Bitrix24.

## Что уже подключено

Приложение использует реальные поля сделки:

```js
UF_CRM_1743501476 // Адрес объекта инфо
UF_CRM_1742459776 // Наименование ИП инфо
UF_CRM_1743669674 // Ответственное лицо ИП инфо
UF_CRM_1707153439 // Город, список
UF_CRM_1747301550750 // Город, текст
```

Основной источник данных — инфоблок `115` "Адрес объекта". В нем есть связи:

```js
PROPERTY_951 // Компания
PROPERTY_959 // Город
PROPERTY_907 // Наименование ИП, ссылка на инфоблок 109
PROPERTY_905 // Ответственное лицо, ссылка на инфоблок 117
```

## Как работает

1. Приложение загружает сделку по ID.
2. Берет клиента из `COMPANY_ID`.
3. Загружает из инфоблока `115` только адреса этого клиента.
4. При выборе адреса подтягивает город, ИП и ответственное лицо.
5. По кнопке "Сохранить в сделку" обновляет сделку через `crm.deal.update`.

## Локальный запуск

Для локального теста используется proxy-сервер, чтобы вебхук не попадал во frontend-код.

```powershell
$env:BITRIX_WEBHOOK_URL = "https://your-domain.bitrix24.kz/rest/user/webhook/"
$env:PORT = "4176"
node server.js
```

После запуска:

```text
http://localhost:4176/?dealId=721361
```

## Встраивание в Bitrix24

Для настоящего встраивания нужен не входящий вебхук, а локальное приложение Bitrix24 с правами:

- `crm`
- `lists`
- `placement`

Приложение встраивается в карточку сделки как пользовательский тип CRM-поля, а не как отдельная вкладка.

Для локального приложения укажите:

```text
URL приложения: https://bitrix24-dependent-fields.vercel.app/
URL установки: https://bitrix24-dependent-fields.vercel.app/install.html
```

После установки `install.html`:

1. зарегистрирует пользовательский тип поля `depfields`;
2. создаст поле сделки `UF_CRM_DEP_WIDGET`;
3. добавит это поле в секцию `Зависимые поля` карточки сделки.

## Карточка сделки

Через REST добавлен персональный блок `Зависимые поля` в "Мой вид карточки" пользователя `1143` для воронки сделок `69`. В блок добавлены реальные поля хранения:

- Адрес объекта инфо
- Наименование ИП инфо
- Ответственное лицо ИП инфо
- Город
- Город *
